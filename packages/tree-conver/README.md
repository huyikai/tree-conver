# Tree-Conver

## Description

Tree data conversion tool
## Usage

```shell
npm i tree-conver
```

```javascript
// Import default export
import treeToArray from "tree-conver"
// or Import Deconstruct Export
import { treeToArray, arrayToTree } from "tree-conver"

// Usage
const tree = [...]
const array = [...]
treeToArray(tree)
arrayToTree(array)
```
